Overview
A MySQL-based hospital management system to streamline hospital operations and improve patient care. 
It efficiently manages patients, doctors, nurses, appointments, admissions, wards, and billing, 
ensuring secure, accurate, and accessible healthcare data.


Features
Manage patient records with multiple contacts.
Track doctors and nurses, their specialties, and duty hours.
Schedule and monitor appointments.
Handle admissions and ward details efficiently.
Generate and manage patient bills.



Database: MySQL 10g is used for this database project
